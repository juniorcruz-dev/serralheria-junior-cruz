# Serralheria Junior Cruz — Portões e Automação

Site institucional desenvolvido para divulgação de serviços de
instalação, manutenção e automação de portões residenciais e comerciais.

Este projeto tem como objetivo apresentar:
- serviços oferecidos
- informações claras para o cliente
- estrutura básica em HTML e CSS
- comunicação direta e objetiva

Serviços apresentados:
- instalação de portões
- motores de portão
- manutenção e regulagens
- automatização de portões
- atendimento residencial e comercial

As imagens e conteúdos finais serão adicionados
após alinhamento e validação com o cliente.
