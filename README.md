# Serralheria Junior Cruz — Portões e Automação

Projeto de site institucional desenvolvido para apresentar serviços de
instalação, manutenção e automação de portões residenciais e comerciais.

O objetivo deste projeto é demonstrar:
- apresentação clara dos serviços
- organização das informações
- estrutura HTML e CSS
- comunicação direta com o cliente final

Serviços apresentados no site:
- instalação de portões
- motores de portão
- manutenção e ajustes
- automatização de portões
- atendimento residencial e comercial

As imagens dos trabalhos e conteúdos finais serão adicionados
conforme a evolução do projeto e validação com o cliente.
